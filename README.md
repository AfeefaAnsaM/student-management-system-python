# Student Management System

This project is developed using Python and SQLite as part of my learning and academic practice.

## Objective
To build a simple console-based application to manage student records efficiently.

## Features
- Add student details
- View all student records
- Delete student records
- Data stored using SQLite database

## Technologies Used
- Python
- SQLite

## Project Structure
student_management_system/
├── main.py        # Program execution and menu
├── student.py     # Student-related operations
├── database.py    # Database connection and table creation
└── students.db    # SQLite database file

## How to Run
1. Clone the repository
2. Open terminal in project folder
3. Run:
   python main.py

- Learned SQLite database connectivity
